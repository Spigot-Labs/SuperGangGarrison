how to build

from the repo root:

windows (default packaged platforms):
powershell -ExecutionPolicy Bypass -File .\scripts\package.ps1

linux/macOS (default packaged platforms):
pwsh ./scripts/package.ps1

windows only:
powershell -ExecutionPolicy Bypass -File .\scripts\package.ps1 -Platforms win-x64

windows only with release version:
powershell -ExecutionPolicy Bypass -File .\scripts\package.ps1 -Platforms win-x64 -Version 1.0.1

linux only:
pwsh ./scripts/package.ps1 -Platforms linux-x64

macOS only:
pwsh ./scripts/package.ps1 -Platforms osx-x64,osx-arm64

CI/local bash wrapper:
bash ./scripts/build.sh win-x64

output:
dist\<runtime-id>\
dist\OpenGarrison-*.zip
dist\OpenGarrison-*.tar.gz

notes:
- the package script stages publishes under `dist\_staging` before copying the finished package into `dist\<runtime-id>`
- the package includes client/server content, default config files, `sampleMapRotation.txt`, packaging README, and bundled server plugins
- the package writes `version.txt` at the package root; pass `-Version` for release builds so clients can compare updates reliably
- packaging performs restore/publish only; verification happens separately from this script
